# Quest Command RTS Central Source

This project folder is the central source of truth:

`E:\UnityProjects\QuestCommandRTS`

## Playable Unity assets

- `Assets\Resources\HudArt` contains HUD/card art used by the game.
- `Assets\Resources\Audio` contains weapon, impact, explosion, and voice clips used by the game.
- `Assets\Resources\StructureModels` and `Assets\Resources\UnitModels` contain the in-game Unity model assets.
- `Assets\Scripts\Runtime` contains the merged gameplay source.

## Archived source inputs

- `SourceAssets\CardArt` contains the original Skyraider, Orca Lifter, and Dual Helipad card renders.
- `SourceAssets\CardArt\Structures` contains replacement building menu renders, including the new War Factory photo.
- `SourceAssets\DesktopUnits` contains the tank asset packs from `C:\Users\matth\Desktop\UNITS`.
- `SourceAssets\ReferenceArt` contains loose reference art gathered from the Desktop.
- `SourceAssets\StructureModels\BastionWarFactoryCMV2` contains the new War Factory full Unity package and quick-import static model.
